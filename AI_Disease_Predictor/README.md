# 🧠 AI Disease Prediction System

An AI-powered web application that predicts diseases based on user-entered symptoms using a Machine Learning model built with Flask.

---

## 🚀 Features

- 🧾 Disease prediction based on symptoms
- 📊 AI confidence score
- ⚠ Risk level indicator (Low / Medium / High)
- 📖 Disease description
- 🛡 Precautions suggestions
- 📋 Recent prediction history
- 🎨 Modern UI using Bootstrap
- 🌐 Flask web application
